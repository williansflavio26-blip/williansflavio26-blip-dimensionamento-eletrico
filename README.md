# ⚡ Dimensionamento Elétrico

Este projeto é uma ferramenta interativa desenvolvida para auxiliar no cálculo e dimensionamento de componentes elétricos, utilizando dados técnicos estruturados. 🛠️

## 🚀 Como utilizar
1. No seu computador ou telemóvel, abra o ficheiro `index.html` num navegador (Chrome, Safari, Edge).
2. Introduza os dados solicitados na interface.
3. A aplicação processará os cálculos automaticamente com base na lógica JavaScript e nos dados JSON.

## 🛠️ Tecnologias utilizadas
* **HTML5**: Estrutura da interface do utilizador. 🌐
* **JavaScript (JS)**: Motor de cálculo e interatividade. 🧠
* **JSON**: Base de dados para parâmetros técnicos de dimensionamento. 📁

## 📱 Visualização
O projeto foi desenhado para ser acessível tanto em desktop como em dispositivos móveis.
